# Smart-Agricultural-Machinery-Rental-System-with-ML-Based-Specification-Recommendation
KhetSeva is an Android application designed to simplify agricultural machinery rentals for farmers and machine owners. The platform provides an easy way to list and discover farming equipment through a modern mobile interface.

The project is developed using Kotlin with Jetpack Compose for the frontend and FastAPI for the backend.

🚜 Main Features
Agricultural machinery listings
Machine recommendation system
Machine image upload
Multilingual support
Machine management module
🛠 Tech Stack
Frontend
Kotlin
Jetpack Compose
Retrofit
Backend
Python
FastAPI
Database
SQLite / PostgreSQL

Tractors

Seed Drills
